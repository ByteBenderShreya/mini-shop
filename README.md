# Mini Shop - E-commerce Product Catalog 🛍️

Capstone project for Full-Stack Deployment & Project Architecture.

### **Live Demo:** https://mini-shop-two-alpha.vercel.app/

### **Features:**
- Client-side routing with 3 pages: Home, Products, About
- Modular JavaScript architecture 
- Product data from FakeStore API
- Responsive design
- Optimized & minified assets

### **Tech Stack:** 
HTML5, CSS3, Vanilla JavaScript ES6+, Netlify

### **Routes:**
- `/` - Home page
- `#/products` - Product catalog
- `#/about` - About page

Built for Thiranex Web Dev Capstone
